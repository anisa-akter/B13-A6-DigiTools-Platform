# DigiTools Platform

A modern, responsive digital tools marketplace UI built with React. Users can browse premium products, add items to cart, remove selected items, and clear the cart through a checkout action with instant toast feedback.

## Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- Local JSON-style product dataset

## Core Features

- Product and cart tab toggling with default product view and empty-cart state.
- Real-time cart count in the navbar with add, remove, total calculation, and proceed-to-checkout clear flow.
- Fully responsive landing page sections including hero, stats, products, steps, pricing, CTA, and footer.

## Optional Features Implemented

- Clicking Buy Now increases cart count instantly.
- Buy button briefly changes to Added to cart after adding an item.
- Proceed to Checkout clears all products from cart and resets count.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
